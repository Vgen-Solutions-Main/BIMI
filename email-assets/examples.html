Important Notes for Email Publishing

  • Always use absolute URLs when hosting images (e.g., https://yourdomain.com/images/IBIMjr-400w.png)
  • Include alt text for accessibility and when images don't load
  • Use inline CSS - Email clients don't support external stylesheets
  • Set explicit widths but always include max-width: 100% for mobile
  • Test in multiple email clients (Gmail, Outlook, Apple Mail, etc.)
  • Keep file sizes small for faster loading
